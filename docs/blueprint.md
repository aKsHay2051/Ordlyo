# **App Name**: OrderEase

## Core Features:

- Waitlist Signup: Collect user emails or phone numbers to join the early access waitlist, ensuring only one field is required.
- MongoDB Integration: Store waitlist submissions securely in MongoDB Atlas, handling success and error states gracefully.
- Simple Form Validation: Validate user input to ensure either a valid email or phone number is submitted.
- Optional Feedback: Allow early users to submit feedback through an optional text area.

## Style Guidelines:

- Primary color: Soft green (#A7D1AB), evoking trust and a connection to WhatsApp.
- Background color: Light gray (#F5F5F5), providing a calm and clean backdrop.
- Accent color: Complementary teal (#73A3B1) for call-to-actions and subtle highlights, ensuring contrast without being flashy.
- Body and headline font: 'PT Sans' (sans-serif) to ensure clarity and simplicity, given the non-technical audience. Note: currently only Google Fonts are supported.
- Mobile-first Tailwind CSS layout, focusing on readability and ease of use on smaller screens.
- Use a clear, single-column layout to avoid overwhelming users on mobile devices.
- Subtle transitions and feedback animations for form submissions, maintaining a calm and trustworthy feel.
